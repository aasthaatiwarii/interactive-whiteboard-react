# 🎨 Interactive Whiteboard App

An interactive whiteboard application built with React and HTML5 Canvas that allows users to draw, erase, clear the canvas, undo/redo actions, adjust brush size, change colors, and download their artwork as a PNG image.

## 🚀 Features

* ✏️ Freehand Drawing
* 🎨 Color Picker
* 📏 Adjustable Brush Size
* 🧽 Eraser Tool
* ↩️ Undo Functionality
* ↪️ Redo Functionality
* 🗑️ Clear Canvas
* 📥 Download Canvas as PNG
* ⚛️ Built with React Hooks

## 🛠️ Technologies Used

* React.js
* JavaScript (ES6+)
* HTML5 Canvas API
* CSS
* Vite

## 📦 Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Navigate to the project folder

```bash
cd whiteboard-app
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm run dev
```
